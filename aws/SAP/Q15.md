# SAP 15번 문제

Q15. 회사의 한 보안 정책은 Amazon S3 버킷에서 데이터를 검색하는 모든 이벤트를 기록하도록 요구합니다. 감사 로그를 S3 버킷에 저장해야 합니다. 중앙 집중식 로깅을 위해 지정된 AWS 계정에 감사 로그 S3 버킷을 만들었습니다. S3 버킷에는 쓰기 전용 교차 계정 액세스를 허용하는 버킷 정책이 있습니다. 모든 S3 객체 수준 액세스가 현재 S3 버킷 및 향후 S3 버킷에 대해 기록되어야 합니다.

A. AWS CloudTrail을 활성화하고 감사 로그 S3 버킷을 사용하여 로그 저장 S3 이벤트 소스, 현재 S3 버킷 및 향후 S3 버킷에 대한 데이터 이벤트 로깅을 활성화합니다.

---

모든 이벤트 기록을 위해 계정에 대한 API 호출을 기록하는 AWS CloudTrail을 활성화합니다. CloudTrail의 모든 S3 객체에 대한 데이터 이벤트 로깅을 활성화하면 옵션 지정에 따라 S3 버킷에 대한 데이터 이벤트를 모두 로그할 수 있습니다.
