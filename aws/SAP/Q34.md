# SAP 34번 문제

Q34. 한 회사에서 AWS 조직을 사용하여 1,000개 이상의 AWS 계정을 관리합니다. 이 회사는 새로운 개발자 조직을 만들었습니다. 새 개발자 조직으로 이동해야 하는 540개의 개발자 멤버 계정이 있습니다. 모든 계정은 필요한 모든 정보로 설정되므로 각 계정을 독립 실행형 계정으로 운영할 수 있습니다.
솔루션 설계자가 모든 개발자 계정을 새 개발자 조직으로 이전하기 위해 수행해야 하는 단계는 무엇입니까?

A.

- 관리 계정에서 조직 API의 조직에서 계정 제거 작업을 사용하여 이전 조직의 각 개발자 계정을 제거합니다.
- 새 개발자 기관의 관리 계정에 로그인하고 개발자 계정 마이그레이션의 대상 역할을 하는 자리 표시자 구성원 계정을 만듭니다.
- 새 개발자 조직의 관리 계정에서 조직 API의 조직 간 초대 작업을 호출하여 개발자 계정에 초대를 보냅니다.

---

AWS Organizations는 중앙에서 AWS 계정을 관리할 수 있는 서비스입니다. 모든 개발자 계정이 새 개발자 OU로 이동해야되기 때문에 우선 기존 개발자 계정을 제거합니다. 새 개발자 관리 계정으로 로그인해 Organizations를 생성합니다. 그 다음 대상이 되는 개발자 계정을 초대합니다.
