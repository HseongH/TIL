# SAP 116번 문제

Q116. 한 회사가 AWS에 클라우드 인프라를 보유하고 있으므로 솔루션 설계자는 인프라를 코드로 정의해야 합니다. 인프라는 현재 하나의 AWS 리전에 배포되어 있습니다. 이 회사의 비즈니스 확장 계획에는 여러 AWS 계정의 여러 리전에 배포하는 것이 포함됩니다.
이러한 요구 사항을 충족하기 위해 솔루션 설계자는 무엇을 해야 합니까?

A. AWS Organizations을 사용합니다. 관리 계정에서 AWS CloudFormation 템플릿을 배포합니다. AWS Control Tower를 사용하여 계정 간 배포를 관리합니다.

---

AWS Control Tower는 랜딩 존이라는 안전한 다중 계정 AWS 환경을 설정 및 관리할 수 있는 가장 쉬운 방법을 제공합니다.
